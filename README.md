# Exploratory-Data-Analysis-EDA-
  Business Profit EDA  Python project analyzing monthly business metrics. Key Features :Data Cleaning:Converted messy `"Null"` and `"nun"` text strings into numeric zeros using Pandas. Visualization: Built a Matplotlib bar chart mapping a profit trend from 50 (Jan) to 110 (Nov). nsights:Flagged key data gaps in Apr, Jul, and Dec.
